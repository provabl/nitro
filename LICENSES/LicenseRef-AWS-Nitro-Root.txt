AWS Nitro Attestation PKI Root Certificate
===========================================

The file internal/nsm/AWS_NitroEnclaves_Root-G1.pem is the AWS Nitro Enclaves
Attestation PKI root certificate (G1), published by Amazon Web Services, Inc. at:

    https://aws-nitro-enclaves.amazonaws.com/AWS_NitroEnclaves_Root-G1.zip

It is the public root of trust for AWS Nitro Enclave attestation documents.
Amazon publishes it for the express purpose of incorporation into attestation
verification software (see "Verifying the root of trust",
https://docs.aws.amazon.com/enclaves/latest/user/verify-root.html). It is
vendored here, unmodified, so this tool can verify the certificate chain of a
Nitro attestation document offline.
